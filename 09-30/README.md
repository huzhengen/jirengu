~ 用户目录（我的是windows，是C:\Users\Administrator目录，命令`pwd`可直接查看是具体哪个目录）（在命令行输入`cd ~`进入该目录）

. 当前目录

.. 上一级目录

**背单词**

directory 目录 文件夹

file 文件

make 新建

remove 删除

echo 查找

move 移动

copy 复制

touch 触摸

change 改变

list 罗列

link 链接

find 查找

**缩写**
|中文|英文|缩写|
|:--------:|:-----:|:----:|
| 创建目录 | make directory | mkdir |
| 删除 | remove | rm |
| 移动 | move | mv |
| 复制 | copy | cp |
| 罗列 | list | ls |
| 改变目录 | change directory | cd |
